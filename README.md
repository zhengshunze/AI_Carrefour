# AI_Carrefour

### How to use

- ***For Mac 12.0 up version use belows command to install cocoapods.***
```
sudo gem install cocoapods
pod init
open Podfile
pod install 
```

- if you  mac is under 11.0 version use 
```
sudo gem install -n /usr/local/bin cocoapods -v 1.8.4
```
